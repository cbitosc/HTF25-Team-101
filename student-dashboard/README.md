# Student Dashboard Web App

This is a simple **React web app** to track student academic performance.  
It uses **localStorage** for data storage, so no backend is required.  

## Features
- Add and track **assignments**
- Add and track **exams/grades**
- Dashboard with **average grades** and charts
- Fully **frontend-only** and beginner-friendly

## Installation

1. Clone the repository:

```bash
git clone <repository-url>
cd student-dashboard
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm start
```

The app will run at [http://localhost:3000](http://localhost:3000)

## Tech Stack
- React.js
- Chart.js
- React Router
- LocalStorage
